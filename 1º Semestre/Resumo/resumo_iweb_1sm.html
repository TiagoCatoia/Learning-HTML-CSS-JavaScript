CSS: Plano de Fundo

Definir os efeitos de plano de fundo dos elementos que estão sendo estilizados.

➢ background-color: Define a cor de fundo de um elemento.
➢ background-image: Especifica uma imagem para ser usada como fundo de um elemento.
➢ background-repeat: Controla como a imagem de fundo é repetida ou estendida quando o conteúdo não preenche todo o elemento.
➢ background-position: Define a posição inicial da imagem de fundo dentro do elemento.
➢ background-attachment: Determina se a imagem de fundo rola com o conteúdo ou permanece fixa enquanto o usuário rola a página.

CSS: Modelo de Caixa

Consiste em: margens, bordas, preenchimento 'margin/border/padding' e o conteúdo real.

Estilo de uma borda: 'border-style,border-color,border-width'

border-style:
➢solid: Cria uma borda sólida.
➢dotted: Cria uma borda pontilhada com pequenos pontos.
➢dashed: Cria uma borda tracejada com traços curtos.
➢double: Cria uma borda dupla com duas linhas paralelas.
➢groove: Cria uma borda entalhada, dando a impressão de estar afundada.
➢ridge: Cria uma borda em relevo, dando a impressão de estar elevada.
➢inset: Cria uma borda em alto-relevo, parecendo estar pressionada para dentro.
➢outset: Cria uma borda em baixo-relevo, parecendo estar projetada para fora.
➢none: Remove completamente a borda.
➢hidden: Oculta a borda, mas reserva o espaço que ela ocuparia.
➢ border-width (px, pt, cm, em, etc) e na seguinte ordem: larguras superior, direita, inferior e esquerda, respectivamente.
➢ border-color: red, blue, green, etc…
➢'top,right,bottom,left' ex: border-top-style
➢border-radius: 'borda arredondada'

Algumas destas propriedades também servem para a margin/padding

Essas propriedades podem assumir os seguintes valores:

➢ auto → o browser calcula a margem de forma a centralizar horizontalmente o elemento dentro de seu container.
➢ length → valor em px, pt, cm, etc.
➢ % → especifica uma margem em % da largura do elemento que contém o elemento estilizado.
➢ Inherit → especifica que a margem deve ser herdada do elemento pai.
➢max-width → definir a largura máxima de um elemento

Aula 6 - CSS: Posicionamento

➢ Elementos inline lado a lado, e tais elementos tiverem margens, ele cria espaço suficiente entre os elementos para acomodar as duas margens.
➢ Dois elementos em bloco um em cima do outro, ele junta as duas margens compartilhadas.
O CSS permite alterar o tipo de caixa de um elemento usando a tag display.
→ Se o valor da propriedade display valer block, um elemento inline será transformado em um elemento em bloco.

● A propriedade CSS usada para posicionar elementos HTML em uma página é chamada position: ex: position:relative;

● Existem usualmente os esquemas básicos de posicionamento a seguir:

→ static: Posição padrão, seguindo o fluxo normal do documento.
→ relative: Posição relativa ao seu posicionamento normal, permitindo ajustes com deslocamentos (top, bottom, left, right).
→ absolute: Posição em relação ao elemento pai posicionado de forma relativa ou em relação ao elemento raiz (viewport), sendo removido do fluxo normal do documento.
→ fixed: Posição fixa em relação à janela de visualização (viewport), permanecendo sempre visível mesmo ao rolar a página.
→ sticky: Posição inicialmente estática, mas que se torna fixa em relação à janela de visualização quando o elemento atinge uma determinada posição durante a rolagem.

Flutuação

→Através do conceito de flutuação, uma caixa flutuante pode ser mudada para a esquerda ou direita até que sua borda externa toque a borda da caixa que a contém ou uma outra caixa flutuante.
→Com a flutuação, as caixas são retiradas do fluxo normal.

● A propriedade clear especifica quais elementos podem flutuar ao lado do elemento limpo e de qual lado.

● A propriedade clear pode ter um dos seguintes valores:

➢ none - Permite elementos flutuantes nos dois lados. Isso é o padrão.
➢ left - Não são permitidos elementos flutuantes no lado esquerdo.
➢ right - Não são permitidos elementos flutuantes no lado direito.
➢ both - Não são permitidos elementos flutuantes em ambos os lados.
➢ inherit - O elemento herda o valor de seu ancestral.

Aula 7 - CSS: Flexbox

● Módulo de Layout de Caixa Flexível
● É usado para estruturar layouts de uma maneira mais simples sem o uso de float ou posicionamento.
● Ele provê propriedades usadas para determinar o posicionamento de containers e seus itens.

●Ao aplicar display: flex a um elemento pai (container), você ativa o modelo de layout flexbox para esse container e seus elementos filhos.
➢ Alinhamento dos itens: justify-content e align-items controlam o posicionamento horizontal e vertical dos elementos.
➢ Dimensionamento flexível: flex e suas variações ajustam automaticamente o dimensionamento dos elementos.
➢ Ordenação: order altera a ordem visual dos elementos.
➢ Alinhamento automático de margens: O espaçamento entre os itens é controlado automaticamente com a propriedade margin.
➢ Quebra de linha: flex-wrap organiza os elementos em várias linhas quando há espaço insuficiente.
➢ Esses recursos são habilitados pela propriedade display: flex, oferecendo um layout flexível e responsivo para os elementos dentro de um container.

● A propriedade flex-direction define em que direção o container deseja mostrar os itens flex.
➢ flex-direction: column; para deixar os items do container na verical.
➢ flex-direction: column-reverse items são mostrados na vertical na ordem reversa
➢ flex-direction: row; os itens serão mostrados na horizonatl na ordem normal.
➢ row-reverse; os itens serão mostrados na horizonatl na ordem reversa

● A propriedade flex-wrap define se os itens irão “pular” para a próxima linha ou não, caso a largura do container seja insuficiente para acomodar todos os itens.

● align é usado para alinhar elementos verticalmente
● justify é usado para alinhar elementos horizontalmente
